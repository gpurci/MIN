#  Nature inspired methods

The objective of this project is to address various optimization problems through nature-inspired techniques

---

## Project Structure

| File | Description |
|------|--------------|
| `common` | General-purpose optimization algorithms. |
| `info` | Articles, books etc. |
| `projects` | Projects using nature-inspired optimization methods. |
| `extern_command.cmd` | Optional file that allows stopping or controlling the algorithm externally. |
| `imshow.py` | Display image function. |
| `reshape.py` | Reshape image function. |
| `sys_function.py` | Module function update routine. |
